# 笔记
## 一、Linux的使用
### 查看 & 切换目录
pwd:看当前在哪个文件夹   
ls:列出当前目录里的文件   
cd 文件夹名:进入文件夹
### 文件操作
cat 文件名:直接查看文件内容（看 flag 必备）   
more 文件名 / less 文件名:分页看大文件   
head 文件名:看文件前几行   
tail 文件名:看文件最后几行  
cp 原文件 目标位置:复制文件  
mv 原文件 新位置/新名字:移动 / 重命名  
rm 文件名:删除文件  
mkdir 文件夹名:新建文件夹  
### 查找文件 & 内容
find / -name "flag*":全盘找名字带 flag 的文件  
grep "关键词" 文件名:在文件里搜关键词  
-r :递归在整个目录里搜   
### 权限
chmod +x 脚本名:给文件加执行权限  
chmod 777 文件名:全开权限（随便读写执行）  
whoami:看当前用户是谁  
id：看用户权限信息
## 二、Http协议
浏览器 ↔ 服务器 之间传输网页、数据的规则  
核心请求方法：  
GET  
从服务器拿数据  
参数直接写在 URL 里：?name=xxx&age=18  
能被看到、能收藏、能刷新  
POST  
向服务器提交数据（登录、上传）  
参数在请求体里，不在 URL  
相对更隐蔽  
常见状态码：  
200 OK：正常  
302 / 301：跳转  
404 Not Found：页面不存在  
403 Forbidden：不让你访问  
500 服务器内部错误  
### 三、php基础
运行在服务器上的脚本语言，用来动态生成网页、处理登录、判断权限  
基本写法：  
'<?php  

    // 代码写在这里
?>'

1.变量:  
PHP 变量以 $ 开头，不需要声明类型，直接赋值即可  
变量名只能由字母、数字、下划线组成，不能以数字开头。  
2.输出语句：  
echo，print直接输出字符串或变量  
var_dump:输出变量的类型+值  
3.接受浏览器参数：  
接收 GET 参数  
接收 POST 参数  
接收 Cookie 参数  
4.条件判断  
if/else 语句，核心就是判断条件是否成立  
= 是等于（弱比较，不区分类型，比如"1" == 1为 true）  
=== 是全等（强比较，既比值也比类型，"1" === 1为 false）  
5.字符串与拼接  
用 . 拼接字符串  
6.文件操作  
包含文件：把其他 PHP 文件的代码引入当前文件：
include：找不到文件会警告，继续执行  
require：找不到文件会报错，停止执行  
读取文件内容  
显示当前文件源码  
7.常用函数  
isset($_GET['id'])     // 判断参数是否存在  
empty($str)            // 判断字符串是否为空  
strlen($str)           // 计算字符串长度  
md5($str)              // 对字符串进行MD5加密  
sha1($str)             // 对字符串进行SHA1加密  
strpos($str, "flag")   // 查找字符串中是否包含"flag"
